Practice Page: CSS

Styles and Rules

Styles are graphical changes imposed on HTML by CSS. Rules are sets of styles targted at specific HTML elements.

The paragraph above was formatted using a rule containing several styles.

The rule is a set of properties targeted at a specific element or set of elements. In this case the target is all paragraphs directly folowing a h2 header.

The properties of the rule are the stylistic changes to be applied. In this case the properties are a border in dark slate grey, a background in lavender, italic text, and a line width of 700px.

Styles are diverse and flexible. You'll pick up a lot of them through use. Cheatsheets can be used when required.

Functions

@rules

@rules add enhanced functionality to our CSS.

@rules (pronounced "at-rules") are rules that tell the CSS document to behave a specfic way or perform a specific function not directly related to imposing rules on the html. For example rules exist which import an external style sheet or impose a rule only under certain conditions.

The @import @rule imports an additional stylesheet into the CSS document.

The @media @rule allows us to use media quieries to apply CSS only when certain conditions are true. For example it can cause different rules to be invoked according to the size of the browser window. One has been applied to this window; if its width is below 500 pixels the background will turn dark slate grey.

Shorthand Properties

Shorthand properties can hold values for several parameters in a single line. This helps make our code concise.

We've already used a few shorthand properties in this document

The font property can take multiple values. It tells the browser to use one of these fonts in order of preference from left to right, so that if a font is not available the next best one can be used.

The border property can hold values for the border width, pattern and colour, and the fill colour, among other things.

This text box uses a shorthand property called padding which sets aside space between its borders and its contents. The spacing values are listed within the property; top, right, bottom, then left.

This text box uses the background shorthand property to assign a background image, its ofset and repeat pattern. additional attributes can also be assigned. This saves writing seperate lines for each property.
