5. Cross-Axial Alignment: The align-items property.

The 'align-items' property defines how items are aligned on the cross-axis. They can be aligned to the start, end or center cross-axial point of the container, or they can be stretched to fill the container on the cross-axis.

Here are three images at their default sizes, defined by the image files they originated from.No alignment has been defined. Their cross-axial container size is defined by the size of the largest element, which is the cat.

The align-items property is added in the example below. its default value is 'stretch', which stretches all elements to fit the container; the smaller birds stretch to the cross-axial size of the larger cat.

In the example below the alignment is set to flex-start, which causes the container's contents to align to its cross-axial 'start'. The cross-axis is column, so the cross-axial start is the top of the container.

In the next example the alignment is set to flex-end. The container's contents align to its cross-axial 'end'. In this case, that is its bottom.

In this final example the alignment is set to center.
