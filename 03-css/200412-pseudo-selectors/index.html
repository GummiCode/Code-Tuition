CSS: Pseudo-selectors

Targeting specific instances of elements

Introduction

Pseudo-selectors allow us to select specific instances of an element type using customisable rules. They're powerfully selective.

Suppose our document contains mutliple headings followed by paragraphs of text, and we want the final paragraph of each such section to be italicised. How would we go about this?

We've looked at a few methods to do this sort of thing.

Classes are one way of picking out one specific element among a set of the same element type. However classes can be labourious to use. Lots of elements need modifying to include appropriate classes.

Some CSS properties pick out specific elements too, such as children of specific elements or the first instance of an element following another specific element. These methods are useful but they have limited scope.

Pseudo-classes are a CSS tool that we could use to this end. Pseudo-classes add to a property and give it some extra rules to work with, allowing it to pick out individual instances of elements with great specificity based on their 'state'.

There are mutliple benefits to this.

  • It avoids excessive use of classes. This helps keep HTML dry.
  • It allows the use of pre-made stylesheets without modifying our HTML.
  • It's powerfully selective. Pseudo-selector rules can be programmed in myriad ways to pick out precisely the elements we want to traget.
  • It facilitates HTML maintenance. If appropriate pseudo-classes are used then conscientious changes to the HTML will not cause editing errors.

In the following sections I'll test some pseudo-selectors out. Full details can be found in the MDN web docs.

:first-child, :last-child

first-child causes the property to only apply to instances of the element that are a first-child.
last-child targets instances of the element that are a last-child.

Examples:

  • p:first-child would target all p elements that are first children.
  • li:last-child would target all li elements that are last children. These would commonly be the final element in a list.
  • div p:last-child would target the final p element in sets of children, but only if they are children of a div element.

:nth

Targets elements using a numeric sequence

Various pseudo-classes target elements using a mathematical sequence. These include:

  • :nth-child
  • :nth-of-type
  • :nth-last-child
  • :nth-last-of-type
